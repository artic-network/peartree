# About Pearcore Demo

A minimal bar-chart application built as a reference implementation for the **pearcore** framework.

This demo validates that the pearcore framework provides all the building blocks needed to create a complete application:

- Toolbar and status bar
- Side panel with visual controls
- Canvas-based rendering
- File I/O (drag-drop and file picker)
- Image export (PNG)
- Help and About panels
- Dark/light theme toggle
- Settings persistence

## Framework

Built with [pearcore](https://github.com/artic-network/pearcore) — a reusable application framework extracted from [PearTree](https://peartree.live).

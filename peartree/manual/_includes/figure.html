{%- comment -%} figure.html — Render a captioned screenshot or diagram in the manual. Wraps an image in a rounded, outlined panel with optional background colour, constrained to a specified maximum width. Usage: {% include 'figure.html', src: "images/fig1.png", alt: "The main toolbar" %} {% include 'figure.html', src: "images/fig2.png", alt: "Filter box", legend: "The filter box in the toolbar." %} {% include 'figure.html', src: "images/fig3.png", alt: "Palette panel", maxwidth: "480px", bg: "#073642" %} {% include 'figure.html', src: "images/fig4.png", alt: "Full view", maxwidth: "80%", bg: "transparent" %} Parameters: src (required) — path to the image, relative to the page URL alt (required) — alt text for the image (accessibility) legend (optional) — caption text shown below the image maxwidth (optional) — CSS max-width of the panel, e.g. "600px" or "70%" defaults to "100%" bg (optional) — CSS background of the panel, e.g. "#073642" or "rgba(7,54,66,0.6)" or "transparent" defaults to transparent (no fill, outline only) {%- endcomment -%} {%- assign _maxw = maxwidth | default: "100%" -%} {%- assign _bg = bg | default: "transparent" -%}

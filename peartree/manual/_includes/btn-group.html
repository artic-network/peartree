{%- comment -%} btn-group.html — Render a PearTree toolbar button group inline in the manual. Wraps multiple btn.html buttons in a Bootstrap .btn-group span so borders merge and radii are handled automatically by Bootstrap's group CSS. Usage (Jekyll / Eleventy Liquid): {% include btn-group.html ids="btn-back btn-forward" label="Navigate history" %} {% include btn-group.html ids="btn-drill btn-climb btn-home" label="Navigate subtree" %} Parameters: ids (required) — space-separated list of button ids label (optional) — aria-label for the group (defaults to "Button group") {%- endcomment -%} {%- assign _items = ids | split: " " -%} {%- assign _label = label | default: "Button group" -%} {%- for _bid in _items %}{%- include 'btn.html', id: _bid -%}{%- endfor -%}

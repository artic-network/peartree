# About Sealion v1.0.0-alpha1

**Sealion** is a fast, interactive web-based sequence alignment viewer designed for visualizing and exploring large multiple sequence alignments directly in the browser.

## Features

- **High Performance**: Efficiently renders alignments with thousands of sequences
- **Interactive Navigation**: Smooth scrolling and zooming through alignment data
- **Flexible Visualization**: Multiple color schemes, consensus view, and customizable display options
- **Selection Tools**: Select and manipulate rows, columns, and rectangular regions
- **Data Export**: Copy selections as FASTA format for further analysis
- **Tags & Bookmarks**: Mark and annotate sequences and sites of interest
- **Dark Mode**: Eye-friendly interface for extended viewing sessions

## Development

Sealion was designed by **Andrew Rambaut** - University of Edinburgh

## Technology

Sealion is built with:
- Pure JavaScript with HTML5 Canvas for high-performance rendering
- Bootstrap 5 for responsive UI components
- Marked.js for markdown rendering

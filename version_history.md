## v0.4.0
### Features
- Editing of tip names/dates etc. in the Metadata Table (double click to edit)
- Clade highlighting with various styles
- Import/Export of custom themes
- New colour picker from palettes
- Colour-by for highlighted and collapsed clades
- Multiple example trees offered
- Export and import of clade highlighting and collapsing in tree files.
- Allow rotated selection box parallel to regression line if option is pressed.
- Show +/- 2 stdev of residuals lines in RTT plot
- Confidence interval of RTT regression
- Colour by the residuals or z-score of the tips
- Options for colour palette ranges
- Click on discrete category in legend to select tips (and vice versa)
- Pick a 'label' annotation when exporting trees
- Unhiding - unhidden tips are selected
- Branch labels
- Copy to clipboard buttons in info boxes

### Fixes
- Cosmetic improvements
- Performance improvements for big trees
